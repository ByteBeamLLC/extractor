
ANZ NUTTAB
==========

Description: ~2,600 foods - Food Standards Australia NZ (manual download required)

Download URL: https://www.foodstandards.gov.au/science/monitoringnutrients/afcd/Pages/downloadableexcelfiles.aspx

Instructions:
1. Visit the URL above
2. Download the dataset files
3. Place them in this directory: /Users/talalbazerbachi/Documents/GitHub/extractor/data/nutrition-datasets

Expected files:
- For FAO/INFOODS: Multiple regional CSV/Excel files
- For ANZ NUTTAB: Excel files from the download page

After downloading, run the ETL scripts to process the data.
